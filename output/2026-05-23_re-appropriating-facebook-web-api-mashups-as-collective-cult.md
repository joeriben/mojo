### Kernthese
Der Artikel untersucht, wie Nutzer*innen und semi-professionelle Entwickler*innen durch das Erstellen von Web-API-Mashups die Plattformordnung von Facebook unterlaufen und wiederaneignen. Er schlägt eine Methode vor, um diesen kollektiven 'kulturellen Imaginären' via vergleichender Inhaltsanalyse von Mashups als mobile digitale Praxis bottom-up auf die Spur zu kommen.
