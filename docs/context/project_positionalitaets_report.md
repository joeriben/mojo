Die Multi-Linsen-Architektur (Thematisch / Disziplinär / Latent + Opus-Synthese) produziert als Nebeneffekt einen Positionalitäts-Report: Wo steht Benjamin mit seinen Publikationen in Bezug zu seinen gefühlten Beheimatungen?

**Why:** Benjamin erkannte (2026-04-10), dass die Spannungen zwischen den Linsen (thematisch marginal aber disziplinär zentral) selbst informativ über seine akademische Positionierung sind.

**How to apply:** Perspektivisch könnte der Report auch auf bereits getrackte Journals angewendet werden, nicht nur auf Kandidaten. UI-seitige Aufnahme neuer Journals ist noch ein offenes Thema.
