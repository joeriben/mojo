$55–95 für 696 Artikel (nur 2026) ist zu viel. Nicht jedes Journal verdient denselben Analyseaufwand.

**Why:** Der Wert eines Opus-Volllaufs ($0.06–0.50 pro Artikel) hängt davon ab, wie wahrscheinlich ein Journal relevante Artikel für Benjamin liefert. MedienPädagogik hat eine andere Trefferquote als AI & Society.

**How to apply:** A/B/C-Tier-System für Journals. A = voller Opus-Scan, B = nur günstiges LLM (DeepSeek/Sonnet), C = nur algorithmisch (Citation-Tracker, Keyword-Match). Tier-Zuordnung aus Scout-Ergebnissen + bisherigen Digest-Verdicts ableitbar.
